基于毫米波的抗环境干扰电子设备检测方法

李晔明, 林宇翔, 董玮
September 2020

摘要

As the volume of electronic devices tends to be miniaturized, malicious electronic devices (e.g., spy camera, tiny bomb initiator) have played an important role in some malicious attacks. However, there is no method that can effectively detect malicious electronic devices yet. To this end, a millimeter-wave (mmWave) based electronic detection system is proposed in this paper. The system consists of three parts which is data collection module, data preprocessing module and deep learning model. Due to the existence of a large number of nonlinear components (e.g., diodes, capacitors) in electronic devices, when an electronic device is irradiated with RF signals, it will generate a special signal containing the characteristics of the electronic device. This signal is called a nonlinear response signal. The detection system detects covert electronic devices efficiently by identifying the feature information. Perform preprocessing on the received signals can effectively separate the high and low frequency bands of the signal in order to facilitate feature extraction. Domain-Adaptation Neural Network (DaNN), which is a deep learning model, has been improved and implemented in this system. DaNN can effectively extract environment-independent features and achieve high-accuracy electronic device detection. The recognition accuracy in the lab environment is up to 99.22%. When the system is applied in simulated usage scenario, the detection accuracy of our system keeps at 98.59%, but the SVM-based method accuracy decreases to 96.04%.
